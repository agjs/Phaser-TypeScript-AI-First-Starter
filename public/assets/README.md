# Static Assets

Place images, spritesheets, audio, fonts, and other static files here. They are
served by Vite at the web root, e.g. `public/assets/foo.png` is available as
`/assets/foo.png`.

Register assets in the appropriate Phaser scene's `preload()` method, typically
`src/runtime/phaser/scenes/BootScene/BootScene.ts`.
